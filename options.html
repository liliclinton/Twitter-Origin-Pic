Twitter Origin Pic Options

Save as filename (Twitter): %picid% %userid% %username% %fullname% %time% %tweettext% %tweetid% %filename%
Any "/" in filename except %fullname% and $tweettext% will be converted into subfolders.
For example: %username%/%filename% will save the pics in a folder named by author.
